# ProofPost

ProofPost is a content-proof backend for the CROO Agent Hackathon. It generates platform-ready content, attaches a proof bundle, and lets a buyer independently re-verify the delivery before payment clears.

## What is included

- `backend/` - TypeScript proof engine, CROO provider, and buyer-side requester harness.
- `docs/` - backend handoff and integration notes.

## Quick start

```bash
cd backend
npm install
npm test
npm run build
```

For frontend integration details, see [docs/FRONTEND_HANDOFF.md](docs/FRONTEND_HANDOFF.md).
